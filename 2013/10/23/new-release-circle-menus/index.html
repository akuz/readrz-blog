New Release: Circle Menus
Oct 23, 2013

Yesterday a new version of the website I was talking about was finally released – horay! It features new section selectors based on Bilevel Partition by Mike Bostock of NY Times. Using two circles, it makes it very easy to mix and match data along different dimensions. Front page looks like a search engine (because it is!) now:


Countries section is selected, showing all news about different countries:


Countries section and Finance “related” section are selected, showing financial news about all countries, sorted by the volume of financial news about each country:
